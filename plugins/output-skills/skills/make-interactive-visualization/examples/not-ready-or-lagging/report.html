What happened

Each node: ~60 s of HTTP 503 after rejoin during upgrade.

scope 3 nodes, ~3 min total · customer Plain T-4006
Why it happened

root server-side back-pressure · not DNS caching alone
Fix

SDK: nearestNode + nodes[]. Fails over on 503. Upstream tracked.

    01

    Shared atomic flag

    Pattern

    3 readers. 1 atomic bool. 1 writer.

      /health    ──┐
              ├──→ read_caught_up : atomic<bool> ←──  refresh()  every 3 s
  503 gate   ──┤
              │
  is_alive() ──┘
    Slowest cache sets recovery time. Not the writer.
    02

    Tick-based refresh hides flap

    Raft

    Refresh every 3 s. Sub-3 s flaps invisible.

      ●─────●─────●─────●──→ t
  0s    3s    6s    9s
        ▲ flap here
        gone by next tick

  if (raft_counter % 3 == 0) refresh()
    Edge-trigger fast state. Don't sample it.
    03

    healthy_read_lag = 1000

    Tuning

    apply_lag > 1000 flips flag to false.

      apply_lag:  0 ──────────│─────────── ∞
              READY      ▲      503
                       1000  (default)

  also flips on:  num_queued_writes > 1000
    Threshold = how much staleness clients tolerate.
    04

    DNS LB is best-effort

    Cloud

    ≥1 TTL of inertia between server and client.

      node /health → poller → DNS A → client TTL → request
                  ~30 s    ~30 s     varies
                  └──── ~60 s total inertia ────┘
    Sub-second failover = SDK retry, not DNS.
    05

    SDK retry beats DNS rotation

    SDK

    DNS: ~60 s. SDK retry: 1 request.

      503 received
        │
        ├─ DNS route:  poll → TTL → ok      ≈ 60 s
        │
        └─ SDK route:  nodes[N+1] → ok          ≈ 1 req
    Retry as close to the caller as possible.
    06

    Body decides, not status

    Method

    JSON 503 body = server rejected. Not network.

      ✗ TCP refused / DNS NXDOMAIN     → blame network

  ✓ { "message": "Not Ready..." }  → blame application
       connection completed.
       server actively rejected.
    Inspect the failure body, not just the status code.
